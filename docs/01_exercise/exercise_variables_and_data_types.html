演習：リスト#

演習1#

値

27.4
29.0
33.8
30.1

出力例

[27.4, 29.0, 33.8, 30.1]
<class 'list'>

# 解答用セル

演習2#

リストの演習1で定義したリストlの要素数を出力せよ

出力例

4

# 解答用セル

演習3#

出力例

33.8

# 解答用セル

演習4#

リストの演習1で定義したリストlに含まれる2つの値29.0と33.8をスライシングを利用して出力せよ

出力例

[29.0, 33.8]
